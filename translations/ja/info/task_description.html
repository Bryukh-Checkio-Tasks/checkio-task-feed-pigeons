私は鳩の一羽に餌をあげ始めました。 一分後に二羽が飛んで来て、さらに一分後には三羽が、 その後に四羽が飛んで来ました(例:1+2+3+4+...)。 餌一つは鳩一羽が一分間食べ続けられる量です。 しかしこの場合、全ての鳩に十分な餌はありません。鳩達は到着するなり餌を食べ始めます。 鳩は食事の辞め時を知らない貪欲な動物です。 もし鳩の餌をN個もっていたら、少なくとも何羽の鳩に小麦の餌を与えられるでしょうか？

入力: 与える小麦の量(正の整数)

出力: 餌を貰える鳩の数(正の整数)

Example: 

checkio(1) == 1
checkio(2) == 1
checkio(5) == 3
checkio(10) == 6

どう使われるか: この問題は、様々状況をモデル化できる事を示します。 もちろん、モデルが示すのは限定的な近似値です。 しかし多くの場合、完璧なモデルは必要としていません。

条件: 0 < N < 105.
